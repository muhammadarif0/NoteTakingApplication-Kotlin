# Note Taking Application (Kotlin)

A professional console-based Note Taking Application developed using Kotlin and Object-Oriented Programming (OOP).

## Description
This project allows users to create, view, update, and delete notes through a simple console interface. It is designed as a university project to demonstrate Kotlin programming and OOP concepts.

## Features
- Create new notes
- View all notes
- Search notes
- Update existing notes
- Delete notes
- Manage notes easily

## Technologies Used
- Kotlin
- IntelliJ IDEA Community Edition
- Console Application
- Object-Oriented Programming (OOP)

## Project Structure
- `src` - Contains the main Kotlin source code
- `Main.kt` - Application entry point
